# Python Basics

Basit Python örnekleri ve temel kavramlar.

## İçerik


- **01_hello.py** - Ilk programımız: Ekrana metin yazdırma
- **02_math.py** - Temel matematiksel işlemler
- **03_loop.py** - Döngüler ile listeyi dolaşma
- **04_function.py** - Fonksiyon tanımlama ve kullanma
- **05_sum.py** - Liste elemanlarını toplama
- **06_if_else.py** - Koşullu ifadeler (if-else)
- **07_string.py** - String (metin) işlemleri
- **08_dict.py** - Sözlük (dictionary) veri yapısı
- **09_range.py** - Range fonksiyonu ile döngü
- **10_append.py** - Liste elemanları ekleme

## Kullanım

Her bir Python dosyasını çalıştırmak için:

```bash
python dosya_adi.py
```

Örneğin:

```bash
python 01_hello.py
python 02_math.py
```

## Gereksinimler

- Python 3.x
